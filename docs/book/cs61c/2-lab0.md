# 环境配置

我用了一个不带图形界面的 Ub18.04 + virtual box 其实就是将 WSL 替换为虚拟机。

虚拟机内安装如下内容：

```
$ sudo apt update
$ sudo apt upgrade
$ sudo apt dist-upgrade
$ sudo apt install build-essential cgdb default-jre gdb git python3 python3-pip valgrind
```

windows 下安装 Python 3.X 和 Java 9+
